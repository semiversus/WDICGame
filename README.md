# Installation and Development

1. Create a virtual environment via `python -m venv .venv`
1. Activate the virtual environment (restart terminal in VSCode, or run `.venv\Scripts\activate` on Windows, or run `. .venv/bin/activate` on Linux)
1. Install requirements and game via `pip install -e .`
1. Run game by calling `game` in the terminal
